1. **初始节点特征**：
    - 每个节点都有一个3维的特征向量
    - 节点0：[1, 0, 0]
    - 节点1：[0, 1, 0]
    - 节点2：[0, 0, 1]
    - 节点3：[1, 1, 1]
2. **GNN层输出特征**：
    - GNN层将3维特征转换为2维特征
    - 输出显示了每个节点的新特征
    - 由于使用了ReLU激活函数，所有负值都被设为0
    - 每个节点的新特征都考虑了其邻居节点的信息

这个简单的GNN实现展示了图神经网络的基本原理：

1. 特征聚合：每个节点收集自己和邻居的特征
2. 特征变换：通过权重矩阵进行线性变换
3. 非线性激活：使用ReLU函数引入非线性
